Listas em HTML5

Listas Ordenadas

  1. Item 1
  2. Item 2
  3. Item 3

A principal vantagem ao utilizar-se a tag ol é a capacidade de inserir elementos na lista
em qualquer posição e a ordem / numeração ser alterada automaticamente.

Em uma lista ordenada podem ser escolhidos o tipo e elemento inicial:

type = "A" start = "1"

  1. Item 1
  2. Item 2
  3. Item 3

type = "a" start = "2"

  1. Item 1
  2. Item 2
  3. Item 3

type = "I" start = "1"

  1. Item 1
  2. Item 2
  3. Item 3

tyoe = "i" start = "1"

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas

  • Item 1
  • Item 2
  • Item 3

As listas não ordenadas podem ter tipos diferentes, assim como as listas ordenadas:

type = "circle"

  • Item 1
  • Item 2
  • Item 3

type = "square"

  • Item 1
  • Item 2
  • Item 3
