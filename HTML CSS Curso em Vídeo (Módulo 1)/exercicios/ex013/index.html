Capítulo 1

Capítulo 1.1

Os estilos inline normalmente são utilizados para aplicar determinadas propriedades em uma ou em um número pequeno de tags.

Capítulo 1.2

Quando essas propriedades tem que ser aplicadas em um grande numero de tags, não é recomendado utilizar-se os estilos inline.

Capítulo 2

Capítulo 2.1

Para determinar o estilo de uma tag inline, basta adicionar " style=" " " ao final da tag de abertura.

Capítulo 2.2

No código acima, foram utilizadas as seguintes propriedades:

  • color
  • background-color
  • font-family
  • font-size
