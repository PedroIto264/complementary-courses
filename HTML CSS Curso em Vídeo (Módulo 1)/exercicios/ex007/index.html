Tags Obsoletas

Elementos utilizados em HTML4 podem estar obsoletos na nova versão, como a tag bgcolor, que foi substituída pelo
"background-color" em CSS.

Além disso, existem tags vindas do HTML4 que foram descontinuadas. algumas dessas tags ainda funcionam, porém,
podem deixar de funcionar a qualquer momento, como a tag "marquee"

Todos esses elementos em HTML que foram descontinuados ou estão obsoletos possuem equivalentes em css e javascript.
O site abaixo é um dos sites onde podem ser encontradas as várias tags obsoletas/descontinuadas.
https://dev.w3.org/html5/pf-summary/obsolete.html
