Formatos

Os principais formatos de vídeos para web são:

  • Mp4
  • WebM
  • m4v
  • OGV

Incompatibilidade

Nem todo navegador reconhecerá todos os formatos de vídeo.
Para resolver esse problema, é necessário utilizar outro método para a inserção de vídeos.

Thumbnail

Custo

Por mais que os vídeos tenham seus tamanhos propriamente reduzidos, a hospedagem local acaba por
consumir muito tráfego de dados, por isso, muitas vezes não é o método ideal para a inserção de vídeos.

Inserindo vídeos do Youtube

Para inserir vídeos do Youtube no seu site, basta entrar no site, basta
abrir o Youtube, escolher um vídeo, clicar em compartilhar e selecionar a opção "incorporar".

O mesmo processo pode ser utilizado para inserir vídeos da plataforma "Vimeo"
