Principais tipos de Formatação.

Negrito / Destaque

Nesta frase temos um termo em negrito utilizando-se a tag b.
A tag b é uma tag não semântica.
Nesta frase temos um termo em destaque utilizando-se a tag strong.
Visualmente ambas as tags têm o mesmo efeito visual, porém a tag strong possui valor semântico.

Itálico / Ênfase

Nesta frase temos um termo em itálico utilizando-se a tag i.
A tag i é uma tag não semântica.
Nesta frase temos um termo em ênfase utilizando-se a tag em.
Visualmente ambas as tags têm o mesmo efeito visual, porém a tag em possui valor semântico.

Texto Marcado

Nesta frase temos um termo marcado utilizando-se a tag mark.
A cor dessa marcação pode ser mudada em uma folha de css ou em um parâmetro
dentro da própria tag.
Com o parâmetro style dentro da tag mark adiciona-se o parâmetro "background-color"
para mudar a cor da marcação do texto.
